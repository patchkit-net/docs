Core Features

PatchKit by default comes with a list of core features that are available for all accounts. You may also be interested in our additional features.

  • Fast GZIP+AES automated binary diffs
  • Customizable launcher included with each application
  • High-speed HTTP downloads
  • Windows, OSX, and Linux releases
  • Basic statistics
  • Basic e-mail support
  • Uploads through the web page and Console Tools and from the inside of the Unity Editor
